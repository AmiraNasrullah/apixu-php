# Apixu PHP

PHP library for [Apixu Weather API](https://www.apixu.com/api.aspx)

## Development

* Requirements:
    * Docker
    * Docker Compose

* Run:
    * cd dev
    * Setup env: docker-compose run dev composer install
    * Open env: docker-compose run dev sh
    * Inside env:
        * cd dev
        * make
